Un ingénieur Systèmes et Réseaux me demande une connexion LDAP 

L'ingé : voici la fonction:

<?php
// $ds doit être une ressource de connexion valide

// $person est un nom ou une partie de nom (par exemple, "Jean")

$dn = "o=My Company, c=US";
$filter="(|(sn=$person*)(givenname=$person*))";
$justthese = array("ou", "sn", "givenname", "mail");

$sr=ldap_search($ds, $dn, $filter, $justthese);

$info = ldap_get_entries($ds, $sr);

echo $info["count"]." entries returned\n";
?> 
Moi : Donc je met ça en début de page et vous vous occuperez de mettre les valeurs correspondantes à votre AD ?
